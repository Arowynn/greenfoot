An empty world that can be used to experiment with code.

To start:
- open Intellij
- Select Tools -> Groovy Console... from the menu
- Select the ScratchPad module when asked
- Copy the first lines from 'examples.groovy' into the Groovy Console and press CTRL+Enter (Cmd+Enter on Mac)
- A greenfoot simulator will be started, click on the 'Run' button
- write some code in the Groovy console and watch what it does by pressing CTRL+Enter (Cmd+Enter on Mac)
